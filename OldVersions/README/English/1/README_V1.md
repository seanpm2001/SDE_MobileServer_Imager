## About

The SDE MobileServer imager is a server imaging tool that allows you to make an image (a configuration of your server in a packaged file) of your server to be sent to and from your desktop computer and your mobile computer (phone, phablet, PDA, etc.)

## Core

This tool is part of the SDE Core, and is written in portable C.
